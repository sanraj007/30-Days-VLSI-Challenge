🚀 30 Days VLSI Coding Challenge

Welcome to my 30-day journey of learning and building digital designs using "Verilog HDL" and "Xilinx Vivado".

Instead of just focusing on theory, I decided to learn through hands-on practice — implementing one concept every day and documenting my progress. This repository reflects my consistency, learning process, and growing understanding of VLSI design.

🎯 Why I Started This Challenge

I wanted to:
- Build a strong foundation in digital electronics  
- Gain practical experience with Verilog  
- Understand the complete design flow (RTL -> Simulation -> Synthesis)  
- Stay consistent and track my daily improvement  
- Create a solid project portfolio for VLSI roles  

🛠 Tools & Technologies

- Verilog HDL  
- Xilinx Vivado    
- GitHub (for documentation and version control)  

⚙️ My Daily Workflow

For each design, I follow a simple and structured process:

1. Write Verilog code  
2. Create a testbench  
3. Run simulation and verify waveforms  
4. Analyze RTL schematic  
5. Perform synthesis and observe hardware mapping  
6. Upload code, results, and documentation to GitHub
