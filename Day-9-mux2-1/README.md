🚀 Day 9: 2:1 Multiplexer

Today, I implemented a 2:1 Multiplexer using Verilog in Vivado.

🔹 Description

A Multiplexer (MUX) is a combinational circuit that selects one of the input signals based on a control signal.

Inputs:
- A  
- B  
- Select (sel)  

Output:
- Y  

🔹 Logic Used
- Data selection using conditional logic  
- AND, OR, NOT (for gate-level implementation)  

🛠 Tools Used
- Verilog HDL  
- Xilinx Vivado  

⚙️ What I Did
- Designed 2:1 MUX using Verilog  
- Implemented using multiple modeling styles  
- Created testbench for verification  
- Performed simulation and waveform analysis  
- Analyzed RTL and synthesized design

- `mux.v`  
- `mux_tb.v`  
- `waveform.png`
